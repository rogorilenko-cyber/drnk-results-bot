# Бот для ответов студентов

Этот бот принимает результаты из Telegram Mini App и отправляет вам отчет.

## Важно

Токен бота нельзя хранить в коде. Используйте переменные окружения.

Если токен уже отправлялся в чат или скриншоты, перевыпустите его в BotFather:

`/revoke`

## Как получить ADMIN_CHAT_ID

1. Запустите бота с переменной `BOT_TOKEN`.
2. Напишите боту `/start`.
3. Бот ответит: `Ваш chat_id: ...`
4. Это число укажите как `ADMIN_CHAT_ID`.

`@username` обычно недостаточно для личных сообщений. Нужен числовой chat_id.

## Переменные окружения

- `BOT_TOKEN` — токен бота из BotFather.
- `ADMIN_CHAT_ID` — ваш числовой chat_id, куда слать отчеты.

## Локальный запуск

```bash
BOT_TOKEN="ваш_новый_токен" ADMIN_CHAT_ID="ваш_chat_id" npm start
```

## Деплой

Подойдет Render, Railway, VPS или любой Node.js-хостинг.

Команда запуска:

```bash
npm start
```

После запуска студент проходит тест в Mini App, а бот отправляет вам:

- имя и username студента;
- результат;
- прошел / не прошел;
- список ошибок;
- правильные ответы по ошибкам.
